# ContadorFX

### Descripción 📝 :
Primera aplicación básica con Java FX, Contador Incrementa y Decrementa el texto de un Label

## Elementos utilizados ⚙️ :

#### Java FX
- Button
- Label
- HBox
- VBox
- Scene
- Stage
